PROJECT OVERVIEW
================================================================================
Artisan is a premium furniture single-page website built with modern web 
technologies. It features a clean, elegant design inspired by luxury furniture 
brands with full responsiveness and interactive elements.


TECHNOLOGY STACK
================================================================================

1. HTML5 (HyperText Markup Language 5)
   - Purpose: Structure and content of the website
   - Version: HTML5 (latest standard)
   - Key Features Used:
     * Semantic HTML elements (section, nav, footer, article)
     * Custom data attributes (data-category, data-price)
     * ARIA labels for accessibility
     * Meta tags for SEO and responsive design

2. CSS3 (Cascading Style Sheets 3)
   - Purpose: Styling and visual presentation
   - Version: CSS3 with modern features
   - Key Features Used:
     * CSS Custom Properties (Variables)
     * Flexbox and Grid layouts
     * Transitions and animations
     * Media queries for responsiveness
     * Box shadows and border-radius
     * Gradient backgrounds
     * Transform effects

3. JavaScript (ES6+)
   - Purpose: Interactivity and dynamic behavior
   - Version: ECMAScript 6+ (Modern JavaScript)
   - Key Features Used:
     * Arrow functions
     * Template literals
     * Const and Let declarations
     * DOM manipulation
     * Event listeners
     * Intersection Observer API
     * Debounce functions
     * Async/await patterns

4. Bootstrap 5.3.0
   - Purpose: Responsive grid system and UI components
   - Version: 5.3.0 (Latest stable)
   - Components Used:
     * Navbar (fixed-top, responsive)
     * Carousel (hero slider)
     * Cards (products, categories, features)
     * Grid system (12-column responsive)
     * Forms (contact form)
     * Buttons and utilities
     * Breadcrumbs
     * Modal components

5. Font Awesome 6.4.0
   - Purpose: Icon library
   - Version: 6.4.0
   - Icon Types Used:
     * Solid icons (fas)
     * Regular icons (far)
     * Brand icons (fab)

FEATURES & FUNCTIONALITIES
================================================================================

1. SMOOTH SCROLLING
   - Technology: JavaScript scrollIntoView() API
   - Behavior: Smooth easing animation
   - Offset: 100px from top (accounts for navbar)
   - Usage: All navigation links and anchor tags
   - Benefit: Enhanced user experience, professional feel

2. RESPONSIVE DESIGN
   - Technology: Bootstrap 5 Grid + CSS Media Queries
   - Breakpoints: 
     * Desktop: > 991px
     * Tablet: 768px - 991px
     * Mobile: < 767px
     * Small Mobile: < 575px
   - Approach: Mobile-first design
   - Benefit: Works perfectly on all devices

3. STICKY NAVBAR
   - Technology: CSS position: fixed + JavaScript scroll listener
   - Effect: Shrinks on scroll (padding reduction)
   - Shadow: Increases on scroll for depth
   - Benefit: Always accessible navigation

4. HERO CAROUSEL
   - Technology: Bootstrap 5 Carousel component
   - Features:
     * 3 slides with auto-play
     * 5-second intervals
     * Fade transition
     * Touch/swipe support
     * Navigation indicators
   - Benefit: Showcases multiple products/messages

5. HOVER EFFECTS
   - Technology: CSS3 transitions and transforms
   - Effects:
     * Image zoom (scale 1.03-1.05)
     * Card lift (translateY -5px to -8px)
     * Icon rotation (10deg)
     * Color transitions
     * Shadow enhancement
   - Duration: 0.3s ease
   - Benefit: Interactive, engaging UI

6. ADD TO CART ANIMATION
   - Technology: JavaScript DOM manipulation
   - Effect:
     * Button color change (green)
     * Text change ("Added!")

7. NOTIFICATION SYSTEM

8. SCROLL ANIMATIONS

PERFORMANCE OPTIMIZATIONS
================================================================================

1. Debounce Function
   - Limits function execution frequency
   - Used for scroll events
   - Improves performance

2. Intersection Observer
   - Efficient scroll detection
   - Better than scroll event listeners
   - Browser-optimized

3. Lazy Loading
   - Images load on demand
   - Reduces initial page weight
   - Faster first paint

4. CSS Hardware Acceleration
   - Transform and opacity animations
   - GPU-accelerated
   - Smooth 60fps animations

5. Event Delegation
   - Efficient event handling
   - Reduced memory usage
   - Better performance

CUSTOMIZATION GUIDE
================================================================================

Changing Colors:
Edit CSS variables in style.css (lines 6-22):
  --primary-brown: #8B6F47;
  --dark-brown: #5C4A32;
  --light-brown: #D4A574;

Changing Fonts:
Edit font-family in style.css (line 28):
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;

Adding Products:
Copy a product card structure in index.html and modify:
  - Image URL
  - Product name
  - Price
  - Rating
  - Data attributes

Changing Carousel Images:
Edit background-image URLs in hero section (index.html lines 60-90)
